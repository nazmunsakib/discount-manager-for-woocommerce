# Discount Manager for WooCommerce - Product Overview

## Project Purpose
A WordPress plugin that provides simple to complex discount rules for WooCommerce stores. This is the core package that enables store owners to create flexible discount systems for their e-commerce operations.

## Key Features and Capabilities
- **Flexible Discount Rules**: Support for both simple and complex discount configurations
- **WooCommerce Integration**: Native integration with WooCommerce platform
- **Core Package Architecture**: Designed as a foundational system for discount management
- **WordPress Plugin Standard**: Follows WordPress plugin development standards and conventions

## Target Users and Use Cases
- **E-commerce Store Owners**: Who need to implement various discount strategies
- **WooCommerce Administrators**: Managing promotional campaigns and pricing rules
- **WordPress Developers**: Extending or customizing discount functionality
- **Marketing Teams**: Creating targeted promotional offers and campaigns

## Value Proposition
Provides a comprehensive discount management system that bridges the gap between simple percentage discounts and complex rule-based pricing strategies, enabling WooCommerce stores to implement sophisticated promotional campaigns without requiring custom development.

## Technical Compatibility
- WordPress 5.0+ compatibility
- WooCommerce 5.0+ integration
- PHP 7.4+ requirement
- Tested up to WordPress 6.8 and WooCommerce 8.5
